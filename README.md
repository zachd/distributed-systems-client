Distributed Systems - Lab 1
=====

These files each send a GET request with a message parameter to a PHP echo script running on localhost.

## PHP Usage
    >php client.php MESSAGE

## Python Usage
    >python client.py
    >ENTER MESSAGE

## Haskell Usage
    >ghci
    >:l client.hs
    >get
    >ENTER MESSAGE

## Server Usage
    >php -S localhost:8000 -t .
